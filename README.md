## 📊 Data Analytics
 
### 🌿 The Kerala Paradox
**Published · ISS Virtual Museum · Spring 2026**
 
Kerala ranks #1 in India for development — yet only 30% of women participate in the workforce.
This project investigates why education alone doesn't close the gender gap,
using UN SDG data and India government datasets.
The research was published in the ISS Virtual Museum and featured on a podcast.

**🟡 In Progress · Internship Project · ISS Summer 2026**
